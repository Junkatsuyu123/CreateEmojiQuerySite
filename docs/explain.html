事前準備

絵文字を追加するインスタンスに、１つアカウントを用意する(以下、このアカウントのことを絵文字垢という)

例ではユーザー名を「add_emojis」としているが、利用者が分かれば名前はなんでもよい

使い方

１．絵文字垢にログインし、ドライブに追加したい画像をアップロードする

２．このサイトでクエリを作成する

ユーザID：登録時に決めた英数字のID(@の後ろについている英数字)

追加する絵文字の個数：追加する絵文字の個数

３．作成したら以下のコマンドを１つずつ入力していく

/*まずCONTAINER_ID を取得*/

sudo docker ps -a

/*DBのコンテナに入る*/

sudo docker exec -it DBのCONTAINER_ID /bin/bash

/*misskeyのDBに接続*/

psql -h localhost -p ポート番号 -U ロール名 -d DB名

/*初期値のまま使用している場合は以下のコマンドで接続できる*/

psql -h localhost -p 5432 -U misskey -d misskey

/*接続できたら作成したクエリをコピペすれば絵文字が追加される*/
